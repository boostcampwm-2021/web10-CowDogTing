# 실행 방법

### Client
- npm install
- npm run starts [ 맥북 https ]
- npm run window-start [ 윈도우 https ]

### Server
- npm install
- npm start

### 주의할 점
- key.pem , cert.pem , client.env , server.env 필요
- m1 에서는 Server의 wrtc install 실패 에러
